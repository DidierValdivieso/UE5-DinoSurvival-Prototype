# DinoSurvival Prototype

An immersive Unreal Engine 5 game project featuring a dynamic ecosystem with unique characters, skill progression, and environmental interactions.

## 🔄 Demo in action

## 🎮 Overview

The Yuvea Project is a comprehensive UE5 game framework that demonstrates advanced gameplay mechanics including:
- Multi-character selection system with detailed preview
- Dynamic skill tree with interactive nodes
- Environmental interaction zones (eating, water)
- Real-time weather management
- Custom animated creatures with specialized behaviors

## 🧱 Core Architecture

### Characters
- **Main Character**: Player-controlled avatar with full animation blueprint
- **Bat Character**: Flying creature with specialized movement
- **Biter Character**: Aggressive ground-based entity
- **Glider Character**: Unique aerial gliding mechanics
- **Small Dinosaur**: Basic AI-driven creature
- **Watcher Character**: Observational environmental entity

### Systems
- **Skill Tree**: Interactive node-based progression system
- **Character Selection**: Widget-based character preview and selection
- **Environmental Zones**: Volume-triggered interactions (eating, water)
- **Weather Management**: Dynamic atmospheric conditions
- **Game Stats**: Real-time player statistics tracking

### UI Components
- Main Menu System
- Settings Panel
- Character Selection Interface
- Skill Tree Visualization
- Game Statistics Dashboard

## 🛠️ Technical Implementation

Built with Unreal Engine 5, featuring:
- Custom C++ character classes with specialized movement
- Animation Blueprints for each creature type
- Blueprint-based UI widgets
- Volume-based environmental triggers
- Dynamic sky and weather systems

## 📁 Project Structure
```
/The_Yuvea_Project
├── /Characters       # Character implementations
├── /Animators        # Animation blueprints
├── /Controllers      # Player and game logic controllers
├── /Widgets          # UI component system
├── /Actors           # Environmental and system actors
└── /Core             # Game mode and instance definitions
```
